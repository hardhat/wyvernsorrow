# Story Tracks and Locations

Each character starts at the **Town** and must navigate through the world to rescue captives and eventually reach the **Demon Gate**.

## Character Tracks (Sequence)

| Track | Character | Sequence |
| :--- | :--- | :--- |
| **Path of Iron** | Swordsman | Town → Forest → Ogre Lair → Castle → Mountain Pass → Cave → Harbor → Ruins → Swamp → Shrine → Tower → Volcano → Sky Peak → Demon Gate |
| **Path of Echoes** | Mage | Town → Forest → Crypt → Tower → Shrine → Ruins → Cave → Harbor → Swamp → Castle → Mountain Pass → Sky Peak → Volcano → Demon Gate |
| **Path of Winds** | Wyvern | Town → Forest → Sky Peak → Volcano → Mountain Pass → Harbor → Cave → Swamp → Ruins → Shrine → Tower → Crypt → Castle → Demon Gate |

## Location Descriptions

### Town
The starting point. Initially empty and quiet. As you rescue captives, it transforms:
- **Merchant**: Always present.
- **Blacksmith (Father)**: Unlocked after Cave. Sharpens blades (Atk boost).
- **Healer (Mother)**: Unlocked after Tower. Restores exhausted heroes.
- **Alchemist (Old Man)**: Unlocked after Ogre Lair/Shrine. Brews resistance potions.
- **Carpenter**: Unlocked after Crypt. Rebuilds town structures for better facilities.

### Forest
*The First Challenge.*
- **Swordsman**: A test of endurance against the Werewolf.
- **Mage**: Sensing the magical disturbance in the brush.
- **Wyvern**: Navigating the dense canopy with superior speed.
- **Captive**: Little Girl.

### Ogre Lair
- **Focus**: Brute force.
- **Swordsman**: A direct duel with the Ogre Boss.
- **Captive**: Old Man (Alchemist).

### Crypt
- **Focus**: Occult and Undead.
- **Mage**: Expelling the Skeletons with holy light.
- **Captive**: Carpenter.

### Tower
- **Focus**: Height and Spells.
- **Wyvern/Mage**: Accessible via flight or teleportation easier than stairs.
- **Captive**: Mother (Healer).

### Cave
- **Focus**: Tight spaces and darkness.
- **Swordsman**: Defending narrow corridors.
- **Captive**: Father (Blacksmith).

### Demon Gate
*The Final Stand.* All paths converge here to face the Demon Lord.
- **Swordsman**: Holds the line.
- **Mage**: Disrupts the Demon Lord's ritual.
- **Wyvern**: Distracts the Lord to open a weakness.
